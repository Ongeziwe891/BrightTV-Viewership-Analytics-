# BrightTV-Viewership-Analytics-
📌 Project Overview

BrightTV’s CEO aims to grow the subscription base for the current financial year. This project analyzes user profiles and viewing behavior to generate actionable insights that support the Customer Value Management (CVM) team in achieving this objective.

The analysis focuses on identifying user trends, consumption patterns, and growth opportunities using data analytics techniques.

🎯 #Objectives#
Analyze user demographics and viewing behavior
Identify key factors influencing content consumption
Recommend content strategies for low-usage periods
Propose initiatives to grow BrightTV’s user base
📂 Dataset Description

The dataset includes:

User Profiles
User ID
Age
Gender
Location (Province)
Viewer Transactions
Viewing date & time
Duration watched
Channel/content type
🛠️ Tools & Technologies
SQL (Data extraction & transformation)
Excel / Power BI (Data analysis & visualization)

📈 Key Insights
1. User & Usage Trends
Peak viewing occurs during evening hours (6PM–11PM)
Majority of users fall within the 18–35 age group
Higher engagement observed in urban provinces
2. Factors Influencing Consumption
Time of day (prime time vs off-peak)
Content type (entertainment vs educational)
User demographics (age, lifestyle)
Accessibility & convenience
3. Low Consumption Strategy

To increase engagement during low-usage periods (e.g., early mornings & midday):

Introduce short-form content (quick shows, highlights)
Promote educational & lifestyle content
Use push notifications & recommendations
Offer scheduled reruns of popular content
4. Growth Initiatives
📢 Targeted Marketing Campaigns (based on user segments)
🎁 Referral Programs & Incentives
📱 Mobile-first content optimization
🤝 Partnerships with local content creators
💸 Flexible subscription pricing (bundles, discounts)
📊 Deliverables
Power BI Dashboard (User & Viewing Insights)
SQL Queries for data preparation
20-minute presentation summarizing insights & recommendations
🚀 Key Recommendations
Focus on personalized content recommendations
Invest in prime-time high-demand content
Improve off-peak engagement strategies
Leverage data-driven marketing
📎 Future Improvements
Integrate real-time analytics
Build recommendation systems (AI/ML)
Include customer churn prediction models
