# BrightTV-Viewership-Analytics-
📌 Case Study Overview

This case study focuses on analyzing BrightTV’s user profiles and viewing behavior to support the company’s objective of growing its subscription base. The analysis aims to provide actionable insights to the Customer Value Management (CVM) team by understanding user trends, content consumption patterns, and opportunities for increasing engagement.

🔍 Approach & Methodology

The case study was conducted using a structured data analysis approach:

Data Preparation: Cleaned and joined user profile and viewership datasets using SQL
Data Analysis: Explored viewing patterns by age, time of day, and content/channel type
Feature Engineering: Created age groups and time buckets (e.g., Morning, Afternoon, Evening, Midnight viewing)
Visualization: Built dashboards in Power BI to identify trends and compare user behavior
📈 Key Insights
Peak viewing time occurs during the evening (6PM–11PM), indicating strong prime-time engagement
Young adults (18–35) are the most active users on the platform
Content consumption is influenced by time of day, user demographics, and content type
Low engagement periods (morning and midday) present an opportunity for targeted content strategies
🧾 Summary of Findings

The analysis revealed clear patterns in user behavior and content consumption, highlighting the importance of time-based content strategies and audience segmentation. Recommendations include introducing short-form and engaging content during off-peak hours, leveraging personalized recommendations, and implementing targeted marketing initiatives to drive both user acquisition and retention.
